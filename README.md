# OmniCalc — Premium Scientific Calculator

OmniCalc is a modern, feature-rich scientific calculator designed with a sleek glassmorphic interface. It provides a comprehensive suite of tools for mathematical calculations, function graphing, and unit conversion, all wrapped in a highly polished, responsive web interface.

![Version](https://img.shields.io/badge/version-2.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ Features

### 1. Scientific Calculator
- Full range of trigonometric functions (sin, cos, tan, etc.)
- Logarithmic and exponential functions
- Factorial, absolute value, and rounding
- Memory operations (MC, MR, M+, M-, MS)
- Degrees/Radians toggle
- Responsive keypad with visual feedback

### 2. Graphing Engine
- Plot multiple equations simultaneously
- Interactive canvas with pan and zoom capabilities
- Coordinate tracking on mouse hover
- Support for complex expressions like `sin(x)`, `x^2`, `abs(x)`, etc.

### 3. Unit Converter
- 8 Categories: Length, Mass, Temperature, Area, Volume, Speed, Time, and Data
- Bidirectional conversion with instant updates
- Intuitive category selection and unit swapping

### 4. Advanced UI/UX
- **Multiple Themes:**
  - **Astro Dark:** Glassmorphic deep space (Default)
  - **Neo Light:** Clean clinical slate
  - **Cyberpunk Neon:** High-contrast vaporwave grid
- **Calculation History:** Slide-out sidebar to track and reuse previous results
- **Keyboard Shortcuts:** Full keyboard support for faster input
- **Responsive Design:** Optimized for both desktop and mobile devices

## 🛠️ Technology Stack

- **Frontend:** HTML5, CSS3 (Custom Properties, Flexbox, Grid)
- **Scripting:** Modern JavaScript (ES6 Modules)
- **Graphics:** HTML5 Canvas API
- **Assets:** Google Fonts (Outfit, IBM Plex), FontAwesome 6

## 🚀 Getting Started

No installation or build process is required. Since OmniCalc is built with pure web technologies:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/scientific-calculator.git
   ```
2. Navigate to the project folder:
   ```bash
   cd scientific-calculator
   ```
3. Open `index.html` in any modern web browser.
